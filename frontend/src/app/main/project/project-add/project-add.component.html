Crear un proyecto

Debes ingresar un nombre del proyecto.
El nombre del proyecto debe tener 4 carácteres como minimo.
Debes ingresar una Descripción del proyecto.
La descripción del proyecto debe tener 4 carácteres como minimo.
La descripción del proyecto debe tener 100 carácteres como máximo.
La fecha de inicio no debe estar despues de la fecha de cierre.
